Alice’s Print Service
Time Limit: 1000MS Memory Limit: 32768KB 64bit IO Format: %I64d & %I64u
Submit

Status

Description
Alice is providing print service, while the pricing doesn’t seem to be reasonable, so people using her print service found some tricks to save money.
For example, the price when printing less than 100 pages is 20 cents per page, but when printing not less than 100 pages, you just need to pay only 10 cents per page. It’s easy to figure out that if you want to print 99 pages, the best choice is to print an extra blank page so that the money you need to pay is 100 × 10 cents instead of 99 × 20 cents.
Now given the description of pricing strategy and some queries, your task is to figure out the best ways to complete those queries in order to save money.

Input
The first line contains an integer T (≈ 10) which is the number of test cases. Then T cases follow.
Each case contains 3 lines. The first line contains two integers n, m (0 < n, m ≤ 10 5 ). The second line contains 2n integers s 1, p 1 , s 2, p 2 , …, s n, p n (0=s 1 < s 2 < … < s n ≤ 10 9 , 10 9 ≥ p 1 ≥ p 2 ≥ … ≥ p n ≥ 0).. The price when printing no less than s i but less than s i+1 pages is p i cents per page (for i=1..n-1). The price when printing no less than s n pages is p n cents per page. The third line containing m integers q 1 .. q m (0 ≤ q i ≤ 10 9 ) are the queries.

Output
For each query q i, you should output the minimum amount of money (in cents) to pay if you want to print q i pages, one output in one line.

Sample Input
1
2 3
0 20 100 10
0 99 100

Sample Output
0
1000
1000

Source
2013 Asia Changsha Regional Contest

打印必须选择页数大于q的方案，如果选择页数刚好大于q的方案（即s[i]<=q<s[i+1]）方案，可以只打印q页，否则必须打印s[i]页（价格为s[i]*p[i]）。后者是区间最小值问题，用二分解决。再与前者比较选较小即可。

#include <cstdio>
#include <algorithm>
#include <iostream>
using namespace std;

typedef long long ll;

const int N = 100010;
int n,m;
ll a[N],s[N],t[N];

int main(){
    int T;
    scanf("%d",&T);
    while (T--){
        scanf("%d%d",&n,&m);
        for (int i=0;i<n;i++){
            scanf("%lld%lld",s+i,t+i);
            a[i] = s[i]*t[i];
        }
        a[n] = 0x3f3f3f3f3f3f3f3fll;
        for (int i=n-2;i>-1;i--){
            a[i] = a[i]<a[i+1]?a[i]:a[i+1];
        }
        for (int i=0;i<m;i++){
            ll q;
            scanf("%lld",&q);
            int ind = upper_bound(s,s+n,q)-s;
            ll ans = t[ind-1]*q;
            if (ans > a[ind]) ans = a[ind];

            printf("%lld\n",ans);
        }
    }
    return 0;
}

City Horizon

Description
Farmer John has taken his cows on a trip to the city! As the sun sets, the cows gaze at the city horizon and observe the beautiful silhouettes formed by the rectangular buildings.

The entire horizon is represented by a number line with N (1 ≤ N ≤ 40,000) buildings. Building i’s silhouette has a base that spans locations Ai through Bi along the horizon (1 ≤ Ai < Bi ≤ 1,000,000,000) and has height Hi (1 ≤ Hi ≤ 1,000,000,000). Determine the area, in square units, of the aggregate silhouette formed by all N buildings.

Input
Line 1: A single integer: N
Lines 2.. N+1: Input line i+1 describes building i with three space-separated integers: Ai, Bi, and Hi
Output
Line 1: The total area, in square units, of the silhouettes formed by all N buildings
Sample Input
4
2 5 1
9 10 4
6 8 2
4 6 3
Sample Output
16
Hint
The first building overlaps with the fourth building for an area of 1 square unit, so the total area is just 31 + 14 + 22 + 23 - 1 = 16.

USACO 2007 Open Silver

矩形切割模板题，代码太容易理解了，就是当前枚举到：前k个矩形划分出来的某一个小块，如果不被后面的矩形切割了，便加入答案，否则继续切割。

#include <stdio.h>  
#include <string.h>  
#include <algorithm>  

using namespace std;  

#define ll long long 

const int N = 40010;  

struct Build
{  
    ll x1,y1;  
    ll x2,y2;  
    ll sum;  

    bool operator<(const Build& b2)const {
        if (y2 != b2.y2)
            return y2<b2.y2;
        if (x2 != b2.x2)
            return x2<b2.x2;
        return x1>b2.x1;
    }
};
Build building[N];

ll n;  

void Cover(ll x1,ll y1,ll x2,ll y2,ll k,ll c){  
    while(k<n&&(x1>=building[k].x2||x2<=building[k].x1||y1>=building[k].y2||y2<=building[k].y1)) k++;
    if(k>=n){  
        building[c].sum+=(x2-x1)*(y2-y1);  
        return;  
    }  
    if(x1<building[k].x1){  
        Cover(x1,y1,building[k].x1,y2,k+1,c);  
        x1=building[k].x1;  
    }  
    if(x2>building[k].x2){  
        Cover(building[k].x2,y1,x2,y2,k+1,c);  
        x2=building[k].x2;  
    }  
    if(y1<building[k].y1){  
        Cover(x1,y1,x2,building[k].y1,k+1,c);  
        y1=building[k].y1;  
    }  
    if(y2>building[k].y2){  
        Cover(x1,building[k].y2,x2,y2,k+1,c);  
        y2=building[k].y2;  
    }  
}  

int main(){  
    ll i;  
    ll a,b,c;  
    while(~scanf("%lld",&n)){  
        for(i=0;i<n;i++){  
            scanf("%lld%lld%lld",&a,&b,&c);  
            building[i].x1=a;  
            building[i].y1=0;  
            building[i].x2=b;  
            building[i].y2=c;  
            building[i].sum=0;  
        }  
        sort(building,building+n);  
        for(i=n-1;i>=0;i--)  
            Cover(building[i].x1,building[i].y1,building[i].x2,building[i].y2,i+1,i);  
        ll ans=0;  
        for(i=0;i<n;i++)  
            ans+=building[i].sum;  
        printf("%lld\n",ans);  
    }  
    return 0;  
}

Socket

拷贝构造函数使用dup函数复用socket文件描述符。
=则要先关闭当前文件，再dup。
setNonBlockingMode基于fcntl来操作socket文件描述符。
isWritable(),isReadable()基于select，支持timeout。select可以从一个集合中选择一个空闲的fd，此处开头三句话是定义、清空、加入，因此集合中只有一个fd。
writeData()、readData()、peekData()都是，先判断可写/读否然后写/读操作，失败抛出异常。

socket里面自带了一些异常信息，因此throw new DlRetryEx(strerror(errno));

excecute()
